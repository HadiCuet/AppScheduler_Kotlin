# App Scheduler for Android

Welcome to the App Scheduler app! This is a simple yet powerful tool that lets you schedule any of your installed apps to launch at a specific time.

### What it can do:
*   **Schedule Apps**: Easily pick any app and a specific date and time for it to open.
*   **Manage Schedules**: View all your upcoming and past schedules in a clean list.
*   **Edit & Cancel**: Change the time of a schedule or cancel it with a single tap.
*   **Works Reliably**: Your schedules are automatically restored even if you restart your phone.
*   **Modern & Easy to Use**: Built with the latest tools for a smooth and simple experience.

### Getting Started

This is a standard Android Studio project. Just open it in Android Studio to build and run the app.

All the latest code and features are on the `main` branch. Happy scheduling!
